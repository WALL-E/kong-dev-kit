# kong-dev-kit

Kong Development, out-of-the-box

* Development environment
  * kong 0.9.5 is installed 
  * PostgreSQL 9.5.5

## Start kong

```
kong start
```

## Running for development

```
cd /root
git clone https://github.com/Mashape/kong.git
cd kong

make dev

make test

make install
```

For more details, please check **kong/README.md**
